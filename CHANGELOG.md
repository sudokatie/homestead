# Changelog

## v0.1.0

Initial release.

### Features

- 32x24 farm grid with tillable soil
- Three maps: farm, town, beach
- Three crops: parsnip (4 days), potato (6 days), cauliflower (12 days)
- Five tools: hoe, watering can, scythe, axe, pickaxe
- Energy system with passout mechanic
- Day/night cycle (6 AM - 2 AM)
- Shop for buying seeds
- Shipping bin for selling crops
- One NPC (Emily) with schedule and friendship
- Save/load via localStorage
- Inventory management
- Canvas-based rendering

### Technical

- 363 tests passing
- TypeScript strict mode
- Next.js 14 with App Router
